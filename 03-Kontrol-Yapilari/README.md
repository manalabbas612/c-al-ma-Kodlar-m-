# Kontrol Yapıları / Control Structures

Bu klasör C# kontrol yapılarını gösteren örnekleri içerir.

This folder contains examples showing C# control structures.

## If-Else Örnekleri / If-Else Examples

Farklı if-else senaryolarını gösteren kapsamlı bir örnek. Bu uygulama:
- Basit if-else yapıları
- Çoklu if-else if zinciri
- İç içe koşullar
- Mantıksal operatörler kullanımını gösterir

A comprehensive example showing different if-else scenarios. This application shows:
- Simple if-else structures
- Multiple if-else if chains
- Nested conditions
- Logical operators usage

### Örnekler / Examples

#### 1. Yaş Kontrolü / Age Check
- Yaşa göre kategori belirleme (reşit/reşit değil/yaşlı)
- Age-based categorization (minor/adult/senior)

#### 2. Not Hesaplama / Grade Calculation
- 0-100 arası nota göre harf notu verme
- Letter grade assignment based on 0-100 score

#### 3. Çift/Tek Sayı Kontrolü / Even/Odd Number Check
- Sayının çift mi tek mi olduğunu belirleme
- Pozitif/negatif/sıfır kontrolü
- Determining if a number is even or odd
- Positive/negative/zero check

### Özellikler / Features

