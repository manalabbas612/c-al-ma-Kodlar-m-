# Temel Örnekler / Basic Examples

Bu klasör C# programlama dilinin temel kavramlarını gösteren örnekleri içerir.

This folder contains examples showing basic concepts of C# programming language.

## HelloWorld

Basit bir "Merhaba Dünya" uygulaması. Bu örnek:
- Konsola çıktı vermeyi
- Kullanıcıdan giriş almayı
- String işlemlerini
- Basik C# sintaksını gösterir

A simple "Hello World" application. This example shows:
- Console output
- User input
- String operations
- Basic C# syntax

### Çalıştırma / How to Run

```bash
cd HelloWorld
dotnet run
```

Bu örnek çalıştırıldığında:
1. Türkçe ve İngilizce selamlama mesajları gösterir
2. Kullanıcıdan isim girmesini ister
3. Kişiselleştirilmiş bir karşılama mesajı görüntüler

When this example runs:
1. Shows Turkish and English greeting messages
2. Asks user to enter their name
3. Displays a personalized welcome message
