# Matematiksel İşlemler / Mathematical Operations

Bu klasör matematiksel işlemlerle ilgili C# örneklerini içerir.

This folder contains C# examples related to mathematical operations.

## Hesap Makinesi / Calculator

Basit bir konsol hesap makinesi uygulaması. Bu örnek:
- Temel matematiksel işlemleri (toplama, çıkarma, çarpma, bölme)
- Hata yakalama (try-catch)
- While döngüsü kullanımı
- Switch-case yapısı
- Kullanıcı girişi doğrulama gösterir

A simple console calculator application. This example shows:
- Basic mathematical operations (addition, subtraction, multiplication, division)
- Error handling (try-catch)
- While loop usage
- Switch-case structure
- User input validation

### Özellikler / Features

- ✅ Dört temel matematik işlemi
- ✅ Sıfıra bölme kontrolü
- ✅ Geçersiz giriş kontrolü
- ✅ Sürekli kullanım (devam etme seçeneği)
- ✅ Türkçe ve İngilizce arayüz

- ✅ Four basic math operations
- ✅ Division by zero check
- ✅ Invalid input validation
- ✅ Continuous usage (continue option)
- ✅ Turkish and English interface

### Çalıştırma / How to Run

```bash
cd Hesap-Makinesi
dotnet run
```

### Kullanım Örneği / Usage Example

```
Birinci sayıyı girin / Enter first number: 10
İşlem seçin (+, -, *, /) / Choose operation (+, -, *, /): +
İkinci sayıyı girin / Enter second number: 5
Sonuç / Result: 10 + 5 = 15
```
