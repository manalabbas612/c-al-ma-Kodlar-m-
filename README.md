# C# Çalışma Kodlarım (My C# Practice Codes)

Bu depo, C# programlama dili ile yaptığım basit çalışmaları ve örnek kodları içermektedir.

This repository contains simple C# practice projects and example codes.

## İçerik (Contents)

### 📁 01-Temel-Ornekler (Basic Examples)
- **HelloWorld**: Basit "Merhaba Dünya" uygulaması / Simple "Hello World" application

### 📁 02-Matematiksel-Islemler (Mathematical Operations)
- **Hesap-Makinesi**: Dört işlem yapabilen basit hesap makinesi / Simple calculator with four basic operations

### 📁 03-Kontrol-Yapilari (Control Structures)
- **If-Else-Ornekleri**: Farklı if-else senaryoları / Different if-else scenarios

## Nasıl Çalıştırılır (How to Run)

### Tüm sistemi kurmak için / To set up the entire system:
```bash
# Projeyi klonlayın (Clone the project)
git clone https://github.com/manalabbas612/c-al-ma-Kodlar-m-.git

# Proje dizinine gidin (Navigate to project directory)
cd c-al-ma-Kodlar-m-
```

### Herhangi bir örneği çalıştırmak için / To run any example:
```bash
# Örnek dizinine gidin (Navigate to example directory)
cd [örnek-klasörü]

# Uygulamayı çalıştırın (Run the application)
dotnet run
```

### Örnekler / Examples:
```bash
# Hello World örneği
cd 01-Temel-Ornekler/HelloWorld
dotnet run

# Hesap Makinesi
cd 02-Matematiksel-Islemler/Hesap-Makinesi
dotnet run

# If-Else Örnekleri
cd 03-Kontrol-Yapilari/If-Else-Ornekleri
dotnet run
```
